World Wide Web

  • Hyper Text Markup Language is used to construct Web Pages.
  • HTML is made up of:
    • Text content – what you see.
    • Markup – what it looks like / how it is arranged.
    • References - to other files eg. Images and videos.
    • Links - to other pages.
  • Pages are linked by Hyperlinks.
  • Invented in early 1990s – now has 30 billion pages.


Web Pages

Hypertext Markup Language (HTML)

  • Markup is made up of Tags
  • A HTML tag is always contained within “less-than” and ”greater-than” brackets. eg < >
  • Most tags have an Open Tag (eg <b> is the open tag for bold type font) as well as a Close Tag, (eg </b> is the close tag for bold type font).
    • <NAME OF TAG>Contents</NAME OF TAG>
                ↑   Opening tag                      ↑  Closing tag
    • Whole thing is referred to as an ELEMENT (as mentioned below).
  • Void tags (like <br> (creates a line break)) do not require a closing tag.
  • An Element refers to everything within a set of opening and closing tags. See below for example.
  • An Attribute is a property of a HTML element.
    • An Element with and attribute is displayed as: <tag> attr=”value”>contents</tag>
    • <a></a> tag is an “anchor” tag that is used for making a link
    • For example, to set the "href" attribute of an "anchor" tag to my company URL, you would write:
      <a href=http://www.pinkskipbags.com.au>NameTheLink</a>
      This would allow me to create a hyperlink: www.pinkskipbags.com.au

    Examples and other Tags

    • <b></b> tags = BOLD text.
    • <em></em> tags = emphasis / italics text.
    • An example of using the Attribute property: <a href=http://www.reddit.com>ClickMe</a> is used to create the link: ClickMe
      • href (attribute name), www.reddit.com (value), "ClickMe" is the text (content) of the link to be created.
    • The image tag = <img>
      • It is written in the format <img src=”url” alt=”text”>
      • src (attribute = source), alt (attribute = alternate text to be displayed if url cannot be found.
      • Images are a void tag. A void tag has no content and therefore does not have a closing tag.

    • Whitespace
      • <br> tag creates a break in text to appear on separate lines.
        • It has no content so it is a void tag – no closing tag required.
        • It is an “inline” element (see below). <br> Shows where a line ends.
      • Additional spaces: If you try to add more than one space by pressing the spacebar, the additional spaces will be deleted. You need to add &nbsp; to create multiple     s   p   a    c   e    s   between letters or word.s
      • Most Elements are categorized as BLOCK elements or INLINE elements.
        • Block elements usually start and finish with a new line when displayed in a browser.
          Eg: <h1>, <p>, <ul>
        • Inline elements are usually displayed without line breaks.
          Eg: <b>, <td>, <a>, <img>
      • <p>paragraph</p> is a “block” element. It creates a block / box for all the text.
      • <span>text</span> is an inline element. <div>text</div> is a block element.
        • The most common attribute used for both are Style and Class
